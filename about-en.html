ERROR 404

What doesn't exist becomes valuable.

ERROR 404 is born from a simple idea: in a world where everything is mass-produced, true rarity is absence.

We are surrounded by disposable products, infinite collections, and trends that last only weeks. ERROR 404 is the opposite. It is a brand built on the concept of real scarcity, lost identity, and objects that weren't supposed to exist.

Just like the digital error “404 – Not Found”, each piece of the brand represents something that was removed, hidden, or deleted from the system. And it is precisely this absence that gives it value.

The 404 Philosophy

If it doesn't exist, it becomes desired.
If it is rare, it becomes eternal.

Each piece from ERROR 404 is treated as a materialized digital artifact. Something that “shouldn’t be here,” but escaped the system.

That is why every drop is limited. Not for marketing. But by concept.

404 Models. 404 Units Each.
After that… silence.

ERROR 404 is not an eternal brand. It is a finite project, with a beginning, a middle, and an end. The plan is clear:

404 different models
404 units per model
All numbered
Never repeated
Never reissued

When the 404th model is launched, the brand ends. The system shuts down. The directory closes. The story is complete.

From then on, only collector's pieces may exist, released sporadically as echoes of a system that no longer exists. This transforms every piece into a real limited edition, with increasing value over time.

ISSUE_001 — The beginning of the end

The first piece of ERROR 404 is model 001/404. A minimalist white t-shirt with premium 3D silicone printing, numbered and certified.

The front message — “The object was removed from the system due to excessive rarity” — defines the brand's DNA. It’s not just clothing. It’s the first line of code of a universe that will disappear once it reaches the end.
